# Windward School App

A full-stack Flask application designed for Windward School students and teachers.

## Features

*   **Authentication:** Secure login and registration system.
*   **Role-Based Access:** Separate views and permissions for Students and Teachers.
*   **Daily Rotation Checker:** Calculates the specific class rotation (Days 1-6) for any given date in the 2025-26 school year.
*   **Announcements:** Teachers can post announcements; Students can view them.
*   **Teacher Dashboard:** Teachers can view the student roster and assign grades.
